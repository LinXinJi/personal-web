# 按钮 button

> Go to [documentation](https://zensical.org/docs/authoring/buttons/)

## 配置

```toml title="zensical.toml"
[project.markdown_extensions.attr_list]
```

## 示例

### 基本按钮

```markdown title="button.md"
[Subscribe to our newsletter](#){ .md-button }
```

[Subscribe to our newsletter](#){ .md-button }

### 增添样式的按钮

```markdown title="button-primary.md"
[Subscribe to our newsletter](#){ .md-button .md-button--primary }
```

[Subscribe to our newsletter](#){ .md-button .md-button--primary }

### 带图标的按钮

```markdown title="button-icon.md"
[Send :fontawesome-solid-paper-plane:](#){ .md-button }
```

[Send :fontawesome-solid-paper-plane:](#){ .md-button }

### 带图标和样式的按钮

```markdown title="button-icon.md"
[Send :fontawesome-solid-paper-plane:](#){ .md-button .md-button--primary }
```

[Send :fontawesome-solid-paper-plane:](#){ .md-button .md-button--primary }
